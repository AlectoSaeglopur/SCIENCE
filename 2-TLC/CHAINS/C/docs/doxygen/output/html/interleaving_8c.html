Function for block deinterleaving byte-streams.

The the matrix is filled row-wise and then emptied column-wise. In case streams length is larger than R*C, the algorithm is executed in multiple cycles by filling and emptying the matrix completely at each time.

Parameters
[in]inStreaminput stream
[out]outStreamoutput stream
[in]pParamspointer to interleaving parameters structure
Returns
error ID
  • number of cycles needed to process input stream
  • number of missing elements to completely fill the matrix during final cycle
  • number of elements to be skipped for each row

number of input elements to be processed during last cycle

update counter in case of missing elements during final cycle
