Detailed Description

Main file.

Author
Filippo Valmori
Date
26/08/2024

File for running DVB-S simulation containing:

  1. random info bytes generation;
  2. scrambling;
  3. reed-solomon coding;
  4. interleaving;
  5. convolutional coding;
  6. phase modulation (mapper + srrc filtering);
  7. channel corruption.
